# 🕷️ A Python-based web scraping project that extracts and stores data for analysis.

A simple web scraping project built with Python to extract data from websites and save it for analysis.

---

## 📌 About the Project

This project was developed to practice web scraping concepts using Python.  
It collects data from a website and stores it in a structured format (CSV), enabling further analysis.

---

## 🚀 Features

- Extracts data from web pages
- Parses HTML content
- Stores data in `.csv` format
- Simple and clean Python implementation

---

## 🛠️ Technologies Used

- Python
- Requests
- BeautifulSoup
- CSV

## ▶️ How to Run

1. Clone the repository:
git clone https://github.com/GustavoHodas/job-scraper-project.git

2. Install dependencies:
pip install requests beautifulsoup4

3. Run:
python main.py
